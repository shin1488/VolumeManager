# VolumeManager

Windows 기본 볼륨 믹서가 불편해서 만든 **미니 볼륨 조절 앱**이에요.

오디오가 재생 중인 앱만 쏙 골라서 아이콘으로 보여주고, 아이콘을 클릭하면 그 자리에서 바로 볼륨을 조절할 수 있어요. 유튜브 보면서 음악 듣다가 디스코드 알림 소리만 줄이고 싶을 때처럼, 앱마다 볼륨을 따로 만지고 싶은 순간에 딱 맞게 만들었습니다.

> 이 앱은 Claude(Anthropic)와 함께 **바이브 코딩**으로 만들어졌어요. 기획부터 UI, 오디오 API 연동까지 Claude랑 대화하면서 하나씩 다듬어 완성한 프로젝트입니다.

## 이런 분들께 추천해요

- Windows 기본 볼륨 믹서를 열기가 번거로웠던 분
- 앱별 볼륨을 자주 바꾸는데, 매번 설정 앱까지 들어가는 게 귀찮았던 분
- 화면 한 구석에 떠 있으면서 필요할 때만 살짝 쓰는 가벼운 도구를 찾던 분
- 게임, 스트리밍, 음악 감상, 회의를 동시에 하느라 소리가 뒤섞이는 분

## 주요 기능

- **재생 중인 앱만 표시**: 소리를 내고 있는 앱만 아이콘으로 보여줘서 화면이 깔끔해요.
- **아이콘 클릭 → 바로 볼륨 조절**: 아이콘을 누르면 그 앱 전용 볼륨 슬라이더가 팝업으로 뜹니다.
- **앱별 음소거**: 슬라이더 옆 버튼으로 특정 앱만 바로 뮤트할 수 있어요.
- **항상 위 고정**: 다른 창을 띄워도 가려지지 않아서 게임이나 영상 보는 중에도 바로 조작 가능합니다.
- **창 투명도 조절**: 너무 눈에 띄지 않게 반투명으로 만들 수 있어요.
- **다크 테마 플로팅 UI**: 작고 어두운 디자인이라 화면 어디에 둬도 거슬리지 않아요.

## 설치 방법

1. [releases 폴더](./releases)에서 최신 `VolumeManager-x.x.x.msi` 파일을 내려받으세요.
2. 다운로드한 `.msi` 파일을 더블 클릭하면 설치가 진행됩니다.
3. 설치가 끝나면 시작 메뉴에서 **VolumeManager**를 검색해 실행하세요.

> Windows 10/11 전용입니다. macOS나 Linux에서는 동작하지 않아요.

## 사용법

1. 앱을 실행하면 작은 창이 뜨고, 현재 소리를 내고 있는 앱들의 아이콘이 줄지어 보입니다.
2. 조절하고 싶은 앱의 **아이콘을 클릭**하면 볼륨 슬라이더가 열려요.
3. 슬라이더를 드래그해서 원하는 볼륨으로 맞추세요.
4. 슬라이더 옆의 **뮤트 버튼**을 누르면 해당 앱만 음소거할 수 있습니다.
5. 창을 원하는 위치로 **드래그**해서 놓아두면 그 자리에 고정됩니다.
6. 필요하면 **투명도**를 조절해 화면에 더 자연스럽게 녹여낼 수 있어요.

## 자주 묻는 질문

**Q. 아이콘이 안 보여요.**
해당 앱에서 실제로 소리가 재생되고 있어야 목록에 올라옵니다. 음악이나 영상을 재생한 뒤 잠깐 기다려 보세요.

**Q. 볼륨을 바꿨는데 시스템 볼륨이 같이 움직여요.**
VolumeManager는 **앱별 볼륨**을 조절해요. 시스템 전체 볼륨이 낮게 설정돼 있으면 앱 볼륨을 올려도 작게 들릴 수 있어요.

**Q. 창을 닫으면 설정이 사라지나요?**
위치나 투명도 같은 기본 상태는 실행 중에만 유지돼요.

## 만든 이야기

"윈도우 볼륨 믹서 들어갔다 나오는 게 너무 귀찮다"라는 생각 하나로 시작한 개인 프로젝트예요. Claude와 함께 **바이브 코딩**으로 아이디어를 주고받으며, Kotlin + Compose for Desktop 기반으로 만들었습니다. 완벽한 프로덕션 앱이라기보단 제가 매일 쓰려고 만든 도구에 가까워요. 쓰다가 불편한 점이나 아이디어가 있으면 이슈로 남겨 주세요.
